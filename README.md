## 📖 About The Project

This is a clean, modern, and fully responsive landing page for a fictional camping and outdoor adventure booking platform. Built with a strong emphasis on scalable CSS architecture, this project serves as a practical showcase of writing highly maintainable frontend code using the **BEM (Block Element Modifier)** methodology and a modular file structure.

## ✨ Key Technical Features

This project was developed with clean code principles and UI/UX best practices in mind:

*   **BEM Methodology:** Strict adherence to BEM naming conventions for CSS classes, ensuring that the styling is modular, reusable, and easy to maintain as the project scales.
*   **Modular Architecture:** CSS is divided into logical files (`reset.css`, `grid.css`, `fonts.css`, `style.css`, and `responsive.css`) for optimal organization.
*   **CSS Custom Properties:** Extensive use of CSS variables (`:root`) for centralized theme management (colors, fonts, transitions).
*   **Custom Grid System:** Implementation of a bespoke CSS grid layout system (`grid.css`) for flexible and consistent container sizing across different breakpoints.
*   **Fully Responsive:** Fluid design that adapts seamlessly from mobile devices to large desktop screens, utilizing a mobile-first approach.
*   **Interactive UI Elements:** Smooth hover transitions on cards, buttons, and navigation links.
*   **JavaScript Mobile Menu:** A lightweight vanilla JavaScript implementation (`app.js`) to handle the mobile off-canvas navigation toggle.

## 🚀 How to Run

To view this project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/Hosein-raz/camping-web.git
    ```
2.  Open the project folder.
3.  Simply open the `index.html` file in your preferred web browser.

## 📂 Project Structure

```text
camping-web/
├── fonts/          # Local font files (Roboto, Archivo)
├── images/         # High-quality assets and SVG icons
├── scripts/
│   └── app.js      # Mobile menu toggle logic
├── styles/
│   ├── fonts.css   # Font-face declarations
│   ├── grid.css    # Custom responsive container system
│   ├── reset.css   # CSS reset
│   ├── responsive.css # Media queries
│   └── style.css   # Main BEM component styles
└── index.html      # Main markup
```
